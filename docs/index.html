Material Sample c

URI http://rs.tdwg.org/dwc/terms/MaterialSample
Is Defined By http://rs.tdwg.org/dwc/terms/
Description

A physical result of a sampling (or subsampling) event. In biological collections, the material sample is typically collected, and either preserved or destructively processed.

Example 
                      A whole organism preserved in a collection. A part of an organism isolated for some purpose. A soil sample. A marine microbial sample.

Procedure c

URI https://w3id.org/tern/ontologies/tern/Procedure
Description

Procedure - A workflow, protocol, plan, algorithm, or computational method specifying how to make an Observation , create a Sample , or make a change to the state of the world (via an Actuator . A Procedure is re-usable, and might be involved in many Observations , Samplings , or Actuations . it explains the steps to be carried out to arrive at reproducible Results .

Usage Note [Input](#Input) can be used to capture any input values for a procedure, such as the input value of the basal wedge to estimate the basal area per hectare.
Super-classes sosa:Procedure c
Restrictions rdf:value exactly 1 tern:Method c
ssn:hasInput max 1 tern:Input c
data:vocabulary exactly 1

Property Value c

URI https://w3id.org/tern/ontologies/tern/PropertyValue
Description

A property-value pair. Use the schema:name property for the name of the property. If there is an additional human-readable version of the value, put that into the schema:description property.

Example 
                      @prefix tern: <https://w3id.org/tern/ontologies/tern/> .
@prefix schema: <http://schema.org/> .
@prefix data: <http://linked.data.gov.au/def/datatype/> .
@prefix sosa: <http://w3.org/ns/sosa/> .

<example> a tern:PropertyValue ;
	schema:name "AUSPLOTS_RANGELANDS.SITE.PROPERTY" ;
	schema:description "The property name" ;
	sosa:hasResult [
		a data:Text ;
		data:value "Great Victoria Desert Nature Reserve" ;
	] .
Super-classes schema:PropertyValue c
Restrictions schema:description only xsd:string c
schema:name exactly 1 xsd:string c
sosa:hasResult exactly 1 tern:Result c

Sampling c

URI https://w3id.org/tern/ontologies/tern/Sampling
Description

Sampling - An act of Sampling carries out a (Sampling) Procedure to create or transform one or more Samples .

Usage Note Crushing a rock sample in a ball mill. Digging a pit through a soil sequence. Dividing a field site into quadrants. Drawing blood from a patient. Drilling an observation well. Establishing a station for environmental monitoring. Registering an image of the landscape. Sieving a powder to separate the subset finer than 100-mesh. Selecting a subset of a population. SuperSite c

URI https://w3id.org/tern/ontologies/tern/SuperSite
Description

TERN SuperSites are Long Term Ecological Research (LTER) sites co-located with TERN OzFlux eddy covariance towers, which deliver biological and environmental observations of flora, fauna, microbial biodiversity, soils and hydrology. TERN OzFlux delivers micrometeorological observations of energy, carbon and water exchange between the atmosphere and key Australian ecosystems. Each TERN SuperSite is located in a significant Australian biome and the network spans a wide range of environmental conditions. The SuperSite network collects detailed data sets on flora, fauna and biophysical processes from each SuperSite.

Super-classes tern:Site c
Sub-classes tern:AffiliateSuperSite c
